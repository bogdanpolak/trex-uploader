# TRex Fake Server

### Setup

* Install NodeJS
* Open terminal (command prompt):
```sh
cd fake-server
npm install
```

## Start

```
cd fake-server
node server.js
```

## Test

To test fake-server:

- Open file `test.http`
- Press `Send Request` command (**REST Client** extension have to be installed)

# Node JS packages

```
npm init
npm install express multer cors
npm install d3-dsv
```
